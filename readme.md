# 🚗 AutoSmart - Concessionária do Futuro

![Status do Projeto](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)

O **AutoSmart** é uma plataforma experimental de venda de veículos desenvolvida para um projeto acadêmico no SENAI. O objetivo foi criar uma experiência de compra imersiva, unindo inteligência artificial e visualização avançada de produtos.

---

## 🌟 Diferenciais do Projeto

- **🤖 Atendimento via IA:** Interface de chat com foco em atendimento humanizado para auxiliar na escolha do veículo ideal.
- **📦 Catálogo 3D Interativo:** Visualização detalhada dos carros diretamente no navegador.
- **📱 Realidade Aumentada (RA):** Possibilidade de projetar o veículo em tamanho real no ambiente do usuário através de dispositivos móveis.
- **🎨 Design Moderno:** Interface pensada para transmitir tecnologia e sofisticação.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Estrutura e estilização moderna.
- **JavaScript (ES6+):** Lógica de interação e simulação da IA de vendas.
- **Model-Viewer (Google):** Para renderização de modelos 3D e suporte a Realidade Aumentada (WebXR).
- **Formatos 3D:** Uso de arquivos `.glb` otimizados para web.

## 🚀 Como Rodar o Projeto Localmente

Como este é um projeto baseado em tecnologias web front-end, executá-lo é muito simples:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/joaov-batista/autosmart-experience.git
   

2. **Acesse a pasta do projeto:**
   ```bash
   cd autosmart-experience
   

3. **Inicie um servidor local:**
   Para que os modelos 3D carreguem corretamente devido a políticas de CORS, recomenda-se usar um servidor local. Você pode usar a extensão *Live Server* do VS Code ou via terminal com Python:
   ```bash
   # Se tiver Python instalado
   python -m http.server 8000
   

4. **Abra no navegador:**
   Acesse `http://localhost:8000` (ou a porta indicada pelo Live Server).
